How to run:
1. Download and unzip Selenium Standalone Server http://www.seleniumhq.org/download/
2. Download and install jdk
3. Download and install chromedriver.exe
4. Download and install xunit.runner.console https://www.nuget.org/packages/xunit.runner.console
3. To start Selenium Server in cmd type: java -jar {selenium folder}\selenium-server-standalone-3.8.0.jar -port 4444
4. To run test type in cmd: {xunit folder}\xunit.runner.console.2.3.1\tools\net452\xunit.console" Avanade.dll
